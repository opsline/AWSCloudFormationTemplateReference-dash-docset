AWS Systems Manager MaintenanceWindowTask MaintenanceWindowStepFunctionsParameters

The MaintenanceWindowStepFunctionsParameters property type specifies the parameters for execution of the STEP_FUNCTION for a Maintenance Window task in AWS Systems Manager.

MaintenanceWindowStepFunctionsParameters is a property of the Systems Manager MaintenanceWindowTask TaskInvocationParameters property type.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Input" : String,
  "Name" : String
}

YAML

Input: String
Name: String

Properties

Input

The inputs for the STEP_FUNCTION task.

Required: No

Type: String

Update requires: No interruption

Name

The name of the STEP_FUNCTION task.

Required: No

Type: String

Update requires: No interruption
