Amazon EC2 LaunchTemplate CreditSpecification

The CreditSpecification property type specifies the credit option for CPU usage of a T2 instance for an Amazon EC2 launch template.

CreditSpecification is a property of the Amazon EC2 LaunchTemplate LaunchTemplateData property type.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "CpuCredits" : String
}

YAML


CpuCredits: String

Properties

CpuCredits

The credit option for CPU usage of a T2 instance. Valid values include standard and unlimited.

Required: No

Type: String

Update requires: No interruption
