Amazon EC2 Instance SsmAssociations

SsmAssociations is a property of the AWS::EC2::Instance resource that specifies the SSM document and parameter values in AWS Systems Manager to associate with an instance.

Syntax

JSON

{
  "AssociationParameters" : [ Parameters, ... ],
  "DocumentName" : String
}

YAML

AssociationParameters:
  - Parameters
DocumentName: String

Properties

AssociationParameters

The input parameter values to use with the associated SSM document.

Required: No

Type: List of Amazon EC2 Instance SsmAssociations AssociationParameters

DocumentName

The name of an SSM document to associate with the instance.

Required: Yes

Type: String
