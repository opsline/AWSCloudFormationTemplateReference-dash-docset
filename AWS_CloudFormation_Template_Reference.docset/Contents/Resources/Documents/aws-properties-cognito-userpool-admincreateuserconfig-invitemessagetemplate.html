Amazon Cognito UserPool InviteMessageTemplate

InviteMessageTemplate is a subproperty of the Amazon Cognito UserPool AdminCreateUserConfig property that defines the email and SMS invitation message structure of an Amazon Cognito User Pool.

Syntax

JSON

{
  "EmailMessage" : String,
  "EmailSubject" : String,
  "SMSMessage" : String
}

YAML

EmailMessage: String
EmailSubject: String
SMSMessage: String

Properties

EmailMessage

The message template for email messages.

Type: String

Required: No

EmailSubject

The subject line for email messages.

Type: String

Required: No

SMSMessage

The message template for SMS messages.

Type: String

Required: No
