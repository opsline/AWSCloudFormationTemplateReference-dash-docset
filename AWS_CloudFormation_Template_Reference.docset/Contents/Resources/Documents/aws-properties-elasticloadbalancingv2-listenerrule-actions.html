Elastic Load Balancing ListenerRule Actions

Actions is a property of the AWS::ElasticLoadBalancingV2::ListenerRule resource that specifies the actions an Elastic Load Balancing listener takes when an incoming request meets a listener rule's condition.

Syntax

JSON

{
  "TargetGroupArn" : String,
  "Type" : String
}

YAML

TargetGroupArn: String
Type: String

Properties

TargetGroupArn

The Amazon Resource Name (ARN) of the target group to which Elastic Load Balancing routes the traffic.

Required: Yes

Type: String

Type

The type of action.

Valid values: forward

Required: Yes

Type: String
