Amazon EC2 Instance ElasticGpuSpecification

The ElasticGpuSpecification property is part of the AWS::EC2::Instance resource that specifies the type of Elastic GPU. An Elastic GPU is a GPU resource that you can attach to your Amazon EC2 instance to accelerate the graphics performance of your applications. For more information, see Amazon EC2 Elastic GPUs in the Amazon EC2 User Guide for Windows Instances.

Syntax

JSON

{
  "Type" : String
}

YAML

Type: String

Properties

Type

The type of Elastic GPU.

Required: Yes

Type: String

Update requires: No interruption
