Amazon EC2 SpotFleet TargetGroupsConfig

The TargetGroupsConfig property type describes the target groups to attach to a Spot Fleet. Spot Fleet registers the running Spot Instances with these target groups.

TargetGroupsConfig is a property of the Amazon EC2 SpotFleet LoadBalancersConfig property type.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "TargetGroups" : [ TargetGroup, ... ]
}

YAML

TargetGroups:
  - TargetGroup

Properties

TargetGroups

One or more target groups. Duplicates not allowed. For property constraints, see TargetGroupsConfig in the Amazon EC2 API Reference

Required: Yes

Type: List of Amazon EC2 SpotFleet TargetGroup

Update requires: No interruption
