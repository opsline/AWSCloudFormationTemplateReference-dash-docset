AWS CodeBuild Project S3Logs

S3Logs is a property of the AWS::CodeBuild::Project resource that specifies settings for logs generated by an AWS CodeBuild build in an S3 bucket.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Status" : String,
  "Location" : String
}

YAML

Status:  String
Location:  String

Properties

Status

The current status of the S3 logs for a build project. Valid values are:

  • ENABLED: S3 build logs are enabled for this build project.

  • DISABLED: S3 build logs are not enabled for this build project.

Required: Yes

Type: String

Update requires: No interruption

Location

The ARN of an S3 bucket and the path prefix for S3 logs. If your Amazon S3 bucket name is my-bucket, and your path prefix is build-log, then acceptable formats are my-bucket/build-log or aws:s3:::my-bucket/build-log.

Required: No

Type: String

Update requires: No interruption
