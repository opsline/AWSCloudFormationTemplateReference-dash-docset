Amazon EC2 LaunchTemplate InstanceMarketOptions

The InstanceMarketOptions property type specifies market (purchasing) option for instances in an Amazon EC2 launch template.

InstanceMarketOptions is a property of the Amazon EC2 LaunchTemplate LaunchTemplateData property type.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "SpotOptions" : SpotOptions,
  "MarketType" : String
}

YAML


SpotOptions: SpotOptions
MarketType: String

Properties

MarketType

The market type.

Valid values include: spot

Required: No

Type: String

Update requires: No interruption

SpotOptions

The options for Spot Instances.

Required: No

Type: Amazon EC2 LaunchTemplate SpotOptions

Update requires: No interruption
