Amazon Kinesis Data Analytics Application InputLambdaProcessor

The InputLambdaProcessor property type specifies the Amazon Resource Name (ARN) of a Lambda function for preprocessing records in a stream before the SQL code for an Amazon Kinesis Data Analytics application executes.

InputLambdaProcessor is a property of the Kinesis Data Analytics Application Input property type.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "ResourceARN" : String,
  "RoleARN" : String
}

YAML

  ResourceARN: String
  RoleARN: String

Properties

ResourceARN

The ARN of the AWS Lambda function that operates on records in the stream.

Required: Yes

Type: String

Update requires: No interruption

RoleARN

The ARN of the IAM role that is used to access the AWS Lambda function.

Required: Yes

Type: String

Update requires: No interruption
