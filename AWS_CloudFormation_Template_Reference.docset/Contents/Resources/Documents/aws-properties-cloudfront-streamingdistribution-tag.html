Amazon CloudFront StreamingDistribution Tag

The Tag property type specifies key-value pairs for an Amazon CloudFront streaming distribution.

Tag is a property of the AWS::CloudFront::StreamingDistribution resource type.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Key" : String,
  "Value" : String
}

YAML


Key: String
Value: String

Properties

Key

A string that contains Tag key.

Required: No

Type: String

Update requires: No interruption

Value

A string that contains an optional Tag value.

Required: No

Type: String

Update requires: No interruption
