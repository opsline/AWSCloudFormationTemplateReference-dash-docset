AWS Glue Trigger Condition

The Condition property type specifies a condition for an AWS Glue job trigger predicate.

The Conditions property of the AWS Glue Trigger Predicate property type contains a list of Condition property types.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "LogicalOperator" : String,
  "JobName" : String,
  "State" : String
}

YAML

LogicalOperator: String
JobName: String
State: String

Properties

LogicalOperator

The logical operator for the condition.

Valid values: EQUALS

Required: No

Type: String

Update requires: No interruption

JobName

The name of the associated job. It must match the single-line string pattern: [\u0020-\uD7FF\uE000-\uFFFD\uD800\uDC00-\uDBFF\uDFFF\t]*

Required: No

Type: String

Update requires: No interruption

State

The state of the condition.

Valid values: SUCCEEDED

Required: No

Type: String

Update requires: No interruption
