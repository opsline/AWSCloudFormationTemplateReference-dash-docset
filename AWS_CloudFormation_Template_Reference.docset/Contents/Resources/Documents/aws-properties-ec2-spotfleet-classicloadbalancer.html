Amazon EC2 SpotFleet ClassicLoadBalancer

The ClassicLoadBalancer property type specifies a Classic Load Balancer.

ClassicLoadBalancer is a property of the Amazon EC2 SpotFleet ClassicLoadBalancersConfig property type.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Name" : String
}

YAML

Name: String

Properties

Name

The name of the load balancer.

Required: Yes

Type: String

Update requires: No interruption
