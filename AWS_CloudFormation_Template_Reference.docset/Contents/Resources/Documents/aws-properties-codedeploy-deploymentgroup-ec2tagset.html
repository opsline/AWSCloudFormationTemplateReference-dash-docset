AWS CodeDeploy DeploymentGroup EC2TagSet

The EC2TagSet property type specifies information about groups of tags applied to EC2 instances. The deployment group will include only EC2 instances identified by all the tag groups. Cannot be used in the same template as EC2TagFilters.

For information on using tags and tag groups to help manage your Amazon EC2 instances and on-premises instances, see Tagging Instances for Deployment Groups in AWS CodeDeploy in the AWS CodeDeploy User Guide.

EC2TagSet is a property of the AWS::CodeDeploy::DeploymentGroup resource type.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Ec2TagSetList" : [Ec2TagSetListObject, ...]
}

YAML

Ec2TagSetList: 
  - Ec2TagSetListObject

Properties

Ec2TagSetList

A list containing other lists of EC2 instance tag groups. In order for an instance to be included in the deployment group, it must be identified by all the tag groups in the list.

Duplicates are not allowed.

Required: No

Type: List of AWS CodeDeploy DeploymentGroup EC2TagSetListObject

Update requires: No interruption
