AWS CodeDeploy DeploymentGroup EC2TagSetListObject

The EC2TagSetListObject property type specifies lists of EC2 tags.

EC2TagSetListObject is a property of the AWS CodeDeploy DeploymentGroup EC2TagSet property type.

Syntax

To declare this entity in your AWS CloudFormation template, use the following syntax:

JSON

{
  "Ec2TagGroup" : [EC2TagFilter, ...]
}

YAML

Ec2TagGroup: 
  - EC2TagFilter

Properties

Ec2TagGroup

The EC2 tags that are already applied to EC2 instances that you want to include in the deployment group. AWS CodeDeploy includes all EC2 instances identified by any of the tags you specify in this deployment group.

Duplicates are not allowed.

Required: No

Type: List of AWS CodeDeploy DeploymentGroup Ec2TagFilters

Update requires: No interruption
